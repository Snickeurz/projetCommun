Notre produit

Ce site a été produit dans le cadre de la formation MIAGE lors du projet commun

CONTEXTE :
L’application est une application qui permet de mettre en relation les entreprises et les clients (Personne morale ou personne physique) afin de leur permettre d’échanger des documents de type contrat le plus rapidement et d’en garder une trace dans leur système d’informations. La valeur ajoutée sera de créer une application qui doit être facile pour les entreprises à intégrer dans leur système afin de leur faire perdre le minimum de temps dans la configuration de l’outil
